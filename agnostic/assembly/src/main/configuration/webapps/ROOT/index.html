K2 Kernel

You are running the Sakai K2 kernel!

The following URLs may be interesting.

URL Notes
http://localhost:8080/webapp-sample/status.js

Allows you to see what the classloaders are like and allows you to see the loading process of various classes.
http://localhost:8080/webapp-sample/status.jsp?testclass=org.sakaiproject.kernel.api.jcr.JCRService&go=Find# Will load an instance of the JCRService interface that you should see was exported by org.sakaiproject.kernel2.agnostic:kernel:0.1-SNAPSHOT. i.e... the Kernel.
http://localhost:8080/webapp-sample/helloworld.jsp Shows a very simple JSP that uses the kernel.
http://localhost:8080/webapp-sample/hello A sample servlet that uses the kernel: http://localhost:8080/webapp-sample/hello Is bound to a servlet that uses the kernel.
http://localhost:8080/webdav webdav folder is the base of the webdav server, which you can also mount in Finder or Explorer (note no trailing /)
